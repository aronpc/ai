# Referência de Validação Electron

Validação de apps Electron.

## Pré-requisitos

- ELECTRON_MCP_ENABLED=true
- Porta de debug 9222

## Ferramentas

| Ferramenta | Propósito |
|------------|-----------|
| get_electron_window_info | Informações da janela |
| take_screenshot | Captura visual |
| send_command_to_electron | Interações |
| read_electron_logs | Logs do console |

## Comandos

- click_by_text
- click_by_selector
- fill_input
- send_keyboard_shortcut
- eval (JavaScript)
