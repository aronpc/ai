# Referência de Categorias de Melhoria

Categorias de melhoria de código.

## Categorias

### Extensões de Padrão
- CRUD para entidades similares
- Filtros adicionais
- Mais formatos de export

### Oportunidades de Arquitetura
- Novas capacidades a partir de padrões existentes
- Extensões de infraestrutura

### Configuração
- Hardcoded → configurável
- Feature toggles

### Utilitários
- Mais validadores
- Mais formatters
- Helpers relacionados

### Melhorias de UI
- States faltantes
- Atalhos de teclado

### Infraestrutura
- Extensões de plugins
- Novos tipos de eventos
