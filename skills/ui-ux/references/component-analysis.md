# Referência de Análise de Componentes

Verificações de consistência do design system.

## Verificações de Consistência

- [ ] Botões com mesmo estilo
- [ ] Inputs com mesmo padding
- [ ] Hierarquia tipográfica
- [ ] Espaçamento consistente
- [ ] Cores seguindo o sistema

## Categorias

| Categoria | Exemplos |
|-----------|----------|
| Feedback | Loading, sucesso, erro |
| Navegação | Breadcrumbs, voltar |
| Formulários | Validação, texto de ajuda |
| Layout | Responsivo, espaçamento |
| Acessibilidade | ARIA, contraste |
