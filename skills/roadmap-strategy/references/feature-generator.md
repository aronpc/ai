# Referência de Geração de Features

Brainstorming e priorização de features.

## Framework MoSCoW

| Prioridade | Definição |
|------------|-----------|
| Must | Crítico para MVP |
| Should | Importante mas pode esperar |
| Could | Bom ter |
| Wont | Não planejado |

## Atributos de Feature

- Título
- Descrição
- Justificativa
- Prioridade
- Complexidade
- Impacto
- Dependências
