# Referência de Análise Competitiva

Framework de análise competitiva.

## Processo de Pesquisa

1. Identificar concorrentes (WebSearch)
2. Pesquisar feedback de usuários
3. Extrair pain points
4. Encontrar gaps de mercado

## Estrutura de Output

```json
{
  "competitors": [...],
  "market_gaps": [...],
  "insights_summary": {
    "top_pain_points": [...],
    "differentiator_opportunities": [...]
  }
}
```
