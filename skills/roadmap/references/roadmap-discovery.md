# Referência de Discovery de Roadmap

Processo de discovery para roadmap.

## Fases de Discovery

1. Carregar contexto do projeto
2. Entender propósito
3. Descobrir público-alvo
4. Avaliar estado atual
5. Inferir contexto competitivo
6. Identificar restrições
7. Criar discovery.json

## Estrutura de Output

```json
{
  "project_name": "...",
  "target_audience": {...},
  "product_vision": {...},
  "current_state": {...},
  "competitive_context": {...}
}
```
