# Referência de Auditoria de Acessibilidade

Verificações WCAG 2.2 para auditoria.

## Verificações Rápidas

| Verificação | Requisito |
|-------------|-----------|
| Contraste | Ratio 4.5:1 |
| Teclado | Navegação por tab |
| ARIA | Labels presentes |
| Alt text | Imagens descritas |
| Formulários | Labels associados |
| Foco | Estados visíveis |

## Problemas Comuns

- Indicadores de foco ausentes
- Baixo contraste
- Labels de formulário ausentes
- Sem skip links
- Imagens sem alt text
