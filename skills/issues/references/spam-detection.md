# Referência de Detecção de Spam

Sinais e categorias de spam.

## Categorias de Spam

| Tipo | Sinais |
|------|--------|
| Promocional | Links externos, marketing |
| Abuso | Linguagem ofensiva |
| Gibberish | Texto aleatório |
| Bot | Template, envio em massa |

## Sinais de Detecção

**Alta confiança:**
- Links para produtos não relacionados
- Sem contexto do projeto
- Linguagem de marketing

**Média confiança:**
- Conteúdo muito curto
- Linguagem genérica
- Primeiro post do autor

## Diretrizes

- Nunca fechar automaticamente
- Marcar para revisão humana
- Ser conservador
