# Referência de Classificação de Issues

Categorias e critérios para classificação de issues.

## Categorias Primárias

| Categoria | Descrição | Palavras-chave |
|-----------|-----------|----------------|
| bug | Algo quebrado | error, crash, broken, not working |
| feature | Nova funcionalidade | add, implement, would like |
| documentation | Melhoria de docs | docs, readme, unclear |
| question | Dúvida | how to, help, question |
| duplicate | Repetido | same as |
| spam | Irrelevante | promotional, off-topic |
| feature_creep | Muito escopo | also, and, plus |

## Processo de Classificação

1. Ler título e descrição
2. Verificar palavras-chave
3. Buscar evidências (código, erros)
4. Atribuir categoria
5. Definir nível de confiança
