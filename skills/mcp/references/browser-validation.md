# Referência de Validação Browser

Verificações Puppeteer para web.

## Ferramentas

| Ferramenta | Propósito |
|------------|-----------|
| puppeteer_navigate | Navegar para URL |
| puppeteer_screenshot | Capturar |
| puppeteer_click | Clicar |
| puppeteer_fill | Preencher input |
| puppeteer_evaluate | Executar JavaScript |

## Seletores

1. [data-testid="..."]
2. #id
3. .class
4. input[name="..."]
